# Веб-сервис для управления движением денежных средств (ДДС)

## Описание
Веб-приложение для учета личных и бизнес-финансов с возможностью создания транзакций, управления справочниками и фильтрации записей.

## Функциональность
- CRUD операции для транзакций
- Управление справочниками (статусы, типы, категории, подкатегории)
- Фильтрация транзакций по различным параметрам
- Автоматическая фильтрация зависимых полей (категории зависят от типа, подкатегории от категории)
- Валидация данных на клиенте и сервере

## Технологии
- Python 3.10+
- Django 4.2
- SQLite
- Bootstrap 5
- jQuery (для AJAX-запросов)

## Установка и запуск

1. **Клонирование репозитория**
```bash
git clone git@github.com:Vladdikanov/transaction-service.git
```
2. **Создание виртуального окружения**
```bash
python -m venv venv
venv/Scripts/activate
```
3. **Установка зависимостей**
```bash
cd transaction_project
pip install -r requirements.txt
```
4. **Создание миграций**
```bash
python manage.py makemigrations transaction
```
5. **Применение миграций**
```bash
python manage.py migrate
```
6. **Создание суперпользователя**
```bash
python manage.py createsuperuser
```

7. **Запуск сервера**
```bash
python manage.py runserver
```
